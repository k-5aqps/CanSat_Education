# 🛰️ CanSat 入門教材リポジトリ

## 概要
このリポジトリは、CanSat製作を通して
電子工作・プログラミング・チーム開発を学ぶための教材です。

## ディレクトリ構成
- `docs/` : 教材・解説資料
- `examples/` : サンプルコード
- `assignments/` : 課題・実習内容
- `data/` : サンプルデータ

## 開発環境
- Raspberry Pi Zero
- Python 3.9

## 学習ステップ
1. `examples/01_led_blink/` から順に実行
2. `assignments/` の課題を週ごとに提出
3. `assignments/competition.md`に沿って模擬大会実施/発表会
