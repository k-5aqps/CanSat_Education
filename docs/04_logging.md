# ログ機能の実装
## ログとは？
ログとは，一言で言うと**動作記録**である．CanSatにおいては，各センサから取得したデータ（緯度経度，加速度など）やモータに対する動作指令（進行方向，Duty比など），またCanSatの起動やゴールといった処理を，時系列で記録する．<br>
ログの意義は，異常動作時に発揮される．例えば，何らかの障害（例外）によりシステムがダウンした場合，何の記録もなければ原因究明しようがない．ログは，通常動作の記録のみでなく，異常を検知した場合の記録も重要な役割である．
## loggingモジュール
Pythonではログを出力する強力な標準ライブラリが用意されている．それが，**logging**である．<br>
loggingは以下のような三本柱がある．
|要素|役割|クラス名|
|-|-|-|
|logger|ログ出力命令|getLogger|
|handler|ログ出力先|StreamHandler, FileHandler|
|formatter|ログ書式|Formatter|

それを踏まえて，最もシンプルな標準出力へログを出力する機能を実装したプログラム`examples/o4_logging/logbasic.py`を見てみよう．
